Module

Description

Runtime engine that builds and executes pipelines. Handles source extraction, container image building, and pipeline execution using Docker or Kubernetes.

Responsibilities

  • Extracts and builds source code from zip archives
  • Builds container images with compiled artifacts
  • Executes pipeline work items in containers
  • Supports Docker and Kubernetes execution engines
  • Manages work dependencies and execution order
  • Tracks work results and pipeline status
