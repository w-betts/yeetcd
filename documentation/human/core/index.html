Module

Description

Go-based controller runtime that builds pipelines from protobuf definitions and executes work in containers. Contains pipeline execution engine, condition evaluation, and work orchestration.

graph TD subgraph core["Module: core"] pipeline[pipeline] condition[condition] engine[engine] end pipeline --> condition pipeline --> engine style core fill:#e3f2fd

Responsibilities

  • Load pipeline definitions from protobuf
  • Execute work items in containers (Docker/Kubernetes)
  • Evaluate work conditions at runtime
  • Track work results and status
  • Stream outputs between work items
  • Provide CLI for pipeline operations

Dependencies

  • yeetcd.protocol (protobuf definitions)
  • Docker client
  • Kubernetes client
  • gRPC
