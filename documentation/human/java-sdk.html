Module

Description

Java API for defining continuous deployment pipelines with container-based execution. Provides a fluent builder API for constructing pipeline definitions that are serialized to protobuf format for language-agnostic execution.

Responsibilities

  • Provide fluent API for pipeline definition
  • Define work items and their dependencies
  • Support multiple work definition types (containerized, custom, compound, dynamic)
  • Serialize pipeline definitions to protobuf format
  • Annotation processor generates pipeline definition runner code
  • Manage work context and parameter passing between work items
