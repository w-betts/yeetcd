Description

Core pipeline execution logic. Implements pipeline assembly from protobuf, work execution orchestration, work definitions, conditions, and result tracking.

classDiagram class Pipeline { +Assemble(pb *pb.Pipeline) *Pipeline +Execute(ctx, engine, arguments) error } class WorkResultTracker { +Record(key, result) +Get(key) WorkResult +AlreadyExecuted(key) bool } class Work { +ID string +Definition WorkDefinition +PreviousWork []PreviousWork } class WorkDefinition { <> } Pipeline --> WorkResultTracker Work --> WorkDefinition

Responsibilities

  • Assemble pipelines from protobuf definitions
  • Execute work items with proper dependency ordering
  • Manage WorkResultTracker for idempotent execution
  • Implement work definition types (Containerized, Custom, Compound, Dynamic)
  • Implement conditions (WorkContext, PreviousWorkStatus, And, Or, Not)
  • Handle work output paths and mounts
  • Manage pipeline parameters and arguments
  • Track and report pipeline events and results
