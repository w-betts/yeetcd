Description

CLI entry point for executing pipelines locally using Docker or mock execution engine. Provides 'run' and 'list' commands for pipeline execution.

Responsibilities

  • Provide 'run' command to execute pipelines
  • Provide 'list' command to list available pipelines
  • Initialize Docker or mock execution engine
  • Parse pipeline arguments
  • Report pipeline execution status and results
