Description

Execution engine interfaces and implementations for running containers. Provides the ExecutionEngine interface and concrete implementations for Docker, Kubernetes, and mock execution.

Responsibilities

  • Define ExecutionEngine interface for container execution
  • Provide Docker execution engine implementation
  • Provide Kubernetes execution engine implementation
  • Provide mock execution engine for testing
  • Build container images with artifacts
  • Run jobs in containers with proper resource allocation
  • Manage container lifecycle (start, stop, remove)
  • Stream job stdout/stderr to caller
  • Handle job results and exit codes
