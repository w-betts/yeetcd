Description

Abstract base class for work execution conditions. Provides methods to apply conditions to protobuf builders.

Responsibilities

  • Define base class for all condition types
  • Apply condition to Work protobuf builder
  • Support nested conditions (And, Or, Not)
  • Convert to protobuf Condition format
