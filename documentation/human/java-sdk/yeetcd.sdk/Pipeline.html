Class

Description

Immutable representation of a pipeline definition containing name, parameters, work context, and final work items.

Responsibilities

  • Hold pipeline configuration (name, parameters, work context, final work)
  • Serialize pipeline to protobuf format
  • Manage native work definitions registry
  • Execute native work definitions by pipeline and task name

Implementation Notes

Immutable: all fields are final
Uses static ConcurrentHashMap for native work definitions registry
Built via Pipeline.builder() fluent API
Serializes to PipelineOuterClass.Pipeline protobuf
