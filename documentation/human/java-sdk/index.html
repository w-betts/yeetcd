Description

Java API for defining pipelines with fluent builder pattern. Includes annotation processor that generates protobuf output for pipeline definitions.

Responsibilities

  • Provides fluent API for pipeline definition
  • Defines Work, Pipeline, and WorkDefinition interfaces
  • Implements annotation processor for code generation
  • Supports multiple work definition types (Containerised, Custom, Compound, Dynamic)
  • Provides condition system for controlling work execution
