Description

Condition classes for controlling work execution flow. Conditions determine whether a work item should execute based on work context values or previous work execution status. Supports logical composition (AND, OR, NOT) of conditions.

Responsibilities

  • Define Condition base class for all conditions
  • Implement logical composition conditions (And, Or, Not)
  • Implement work context conditions for key-value matching
  • Implement previous work status conditions
  • Serialize conditions to protobuf format
