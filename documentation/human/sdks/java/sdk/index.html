Module

Description

Java SDK for defining pipelines using a fluent builder API. Provides classes and interfaces for creating pipeline definitions that are compiled to protobuf format at build time.

Responsibilities

  • Provide fluent builder API for pipeline definition
  • Support multiple work definition types (Containerised, Custom, Compound, DynamicWorkGenerating)
  • Generate protobuf pipeline definitions at compile time via annotation processor
  • Implement work context and parameter passing between pipeline stages
  • Support conditional work execution based on context or previous work status

Dependencies

  • yeetcd.protocol (protobuf definitions)
  • Lombok
  • Apache Commons Codec
