Description

Package for conditional work execution. Provides condition types for context-based and status-based conditions, and logical composition.

Responsibilities

  • Define Condition base class
  • Provide WorkContextCondition for context-based execution
  • Provide PreviousWorkStatusCondition for status-based execution
  • Support logical operators (And, Or, Not)
