Continuous Deployment Pipeline System

System Overview

yeetcd is a continuous deployment solution with container-based pipeline execution. It provides a Java SDK for defining pipelines, a controller for execution, and testing utilities for validating pipeline logic.

Architecture

The system follows a modular architecture with clear separation of concerns:

  • Protocol Layer: Language-agnostic protobuf definitions for pipeline serialization
  • SDK Layer: Java API for pipeline definition with fluent builders
  • Controller Layer: Runtime execution engine with Docker/Kubernetes support
  • Test Layer: In-memory testing utilities for pipeline validation
